Frequently Asked Questions

How do I add games to GoGBA?

GoGBA uses your device's file system to access game ROM files. To add games:

  1. Open the App and tap the "Add Games" button
  2. Select "Scan Game Directory"
  3. Choose a folder containing .gba files
  4. The App will automatically scan and add all GBA games in that folder

Where are my save files stored?

Save files (.sav files) are stored locally on your device:

  • Android: External storage directory
  • iOS: App Documents directory

Save files are automatically saved every 10 seconds, when you exit the game, or when the app goes to the background.

Can I use GoGBA with a controller?

Controller support is not currently available in this version of GoGBA. This feature may be added in future updates.

Does GoGBA support cloud saves?

Cloud save functionality is not currently available. All saves are stored locally on your device. We recommend regularly backing up your save files.

How do I change the game speed?

While playing a game, tap the speed button (1x, 2x, or 3x) located in the game interface. You can adjust the speed from 1x (normal) to 3x (fastest).

Can I customize the virtual button layout?

Basic button customization is available. You can adjust button positions, sizes, and transparency through the settings menu. Full customization features are being developed for future updates.

What languages does GoGBA support?

GoGBA supports 24 languages:

  • English
  • 中文 (Simplified Chinese)
  • Español (Spanish)
  • Français (French)
  • Deutsch (German)
  • Português (Portuguese)
  • العربية (Arabic)
  • 日本語 (Japanese)
  • 한국어 (Korean)
  • Русский (Russian)
  • Italiano (Italian)
  • Nederlands (Dutch)
  • Svenska (Swedish)

You can change the language in Settings → Language.

How do I change the theme?

Go to Settings → Display → Theme to choose from available themes:

  • OLED Neon (default)
  • Dracula
  • Nord
  • Catppuccin Mocha
  • Gruvbox

My game crashes or doesn't load. What should I do?

If you encounter crashes or loading issues:

  1. Ensure your ROM file is a valid GBA file (.gba extension)
  2. Try restarting the App
  3. Check that you have sufficient storage space
  4. Update to the latest version of GoGBA
  5. Contact support with details about the issue

Can I play games in both portrait and landscape mode?

Yes! GoGBA supports both portrait and landscape orientations. You can lock the orientation in Settings → Display → Orientation, or toggle it while playing a game through the in-game menu.

How do I clear the cover cache?

To clear cached game covers, go to Settings → Data → Clear Cover Cache. This will remove all downloaded cover images, and they will be re-downloaded when needed.
